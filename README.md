NUMBER GUESSING GAME

OVERVIEW

The Number Guessing Game is a Java-based console application that generates a random number between 1 and 100 and challenges the user to guess it. The application provides interactive feedback after each attempt, indicating whether the guessed number is too high or too low until the correct number is identified.

FEATURES

• Random number generation within a specified range
• Interactive user input through the console
• Input validation and exception handling
• Real-time feedback for each guess
• Attempt tracking and performance monitoring

TECHNOLOGIES USED

• Java
• Visual Studio Code
• Git
• GitHub

PROJECT STRUCTURE

Compile the Program

Execute the Program

java NumberGuessingGame
